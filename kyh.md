# 팀 프로젝트에 대한 피드백

1.어려웠던 점 2. 좋았던 점

협업해서 한 일들이기 때문에 혼자 프로젝트를 수행하는 것보다 빠르고 효율성 있게 역할 분담을 해서 작업을 하였으므로 좋은점이 많았으며 어려웠던 점이라면 많지 않은 지식으로 프로젝트를 수행하다보니 거기에서 오는 한계는 있었으나 혼자가 아니라 여러사람이 해낸 것이므로 빠른 수습과 수정을 통해 팀 프로젝트를 완료할 수 있었다.
